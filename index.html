Optimizing Bitcoin: Cryptocurrency System in Rust
Mar 2020 - May 2020

Language: Rust

The throughput of traditional bitcoin longest chain protocol is limited by the network delay and does not scale well as the speed of transaction generation increases. In this project, we optimized the throughput of bitcoin by decoupling the transaction and proposer blocks. The main chain consists of only proposer blocks, and the mining difficulty for proposer blocks stays the same as the baseline implementaion. The proposer blocks contains references to transaction blocks, which includes the actual transactions. The mining difficulty for transaction blocks is lower so that they can be mined at a faster rate, improving the throughput of the crypto system.

You can find the project here.

DeadDrop: Covert Channel Through Random Number Generator
Oct 2019 - Dec 2019

Languages: C++, assembly

In computer security, a covert channel is a type of attack that creates a capability to transfer information objects between processes that are not supposed to be allowed to communicate by the computer security policy.
I designed and implemented a cross-core contention-based covert channel through hardware random number generator (RNG) using C++ and inline assembly. The channel works by allowing two processes, Trojan and Spy, to transmit bits by contending and probing the shared buffer in hardware RNG. I implemented UART protocol on top and established retransmission to improve the channel accuracy. The bandwidth of the covert channel is estimated to be around 7 kbps.
